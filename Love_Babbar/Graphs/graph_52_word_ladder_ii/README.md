# graph_52_word_ladder_ii

A transformation sequence from word beginWord to word endWord using a dictionary wordList is a sequence of words beginWord -> s1 -> s2 -> ... -> sk such that every adjacent pair of words differs by a single letter. Given two words, beginWord and endWord, and a dictionary wordList, return all the shortest transformation sequences from beginWord to endWord, or an empty list if no such sequence exists. Each sequence should be returned as a list of the words [beginWord, s1, s2, ..., sk].<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/word-ladder-ii/' target='_blank'>LeetCode 126</a>
