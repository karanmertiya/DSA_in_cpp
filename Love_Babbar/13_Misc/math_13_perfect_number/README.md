# Math 13 Perfect Number

## Description
A perfect number is a positive integer that is equal to the sum of its positive divisors, excluding the number itself. A divisor of an integer x is an integer that can divide x evenly. Given an integer n, return true if n is a perfect number, otherwise return false.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/perfect-number/' target='_blank'>LeetCode 507</a>

## Examples
**Example 1:** Sum of divisors.
