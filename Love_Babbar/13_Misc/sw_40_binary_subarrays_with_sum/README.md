# Sw 40 Binary Subarrays With Sum

## Description
Given a binary array nums and an integer goal, return the number of non-empty subarrays with a sum goal.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/binary-subarrays-with-sum/' target='_blank'>LeetCode 930</a>

## Examples
**Example 1:** atMost(goal) - atMost(goal - 1).
