# Math 12 Ugly Number

## Description
An ugly number is a positive integer whose prime factors are limited to 2, 3, and 5. (e.g. 1, 2, 3, 4, 5, 6, 8, 9, 10, 12, ...) Given an integer n, return true if n is an ugly number.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/ugly-number/' target='_blank'>LeetCode 263</a>
