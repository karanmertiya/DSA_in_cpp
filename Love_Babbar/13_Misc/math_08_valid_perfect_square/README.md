# Math 08 Valid Perfect Square

## Description
Given a positive integer num, return true if num is a perfect square or false otherwise. A perfect square is an integer that is the square of an integer. In other words, it is the product of some integer with itself.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/valid-perfect-square/' target='_blank'>LeetCode 367</a>

## Examples
**Example 1:** 
**Input:** num = 16
**Output:** true
