# Math 09 Power Of Three

## Description
Given an integer n, return true if it is a power of three. Otherwise, return false. An integer n is a power of three, if there exists an integer x such that n == 3^x.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/power-of-three/' target='_blank'>LeetCode 326</a>

**Example 1:** Modulo with largest power.
