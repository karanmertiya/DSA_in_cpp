# Ll 18 Swap Nodes In Pairs

## Description
Given a linked list, swap every two adjacent nodes and return its head. You must solve the problem without modifying the values in the list's nodes (i.e., only nodes themselves may be changed.)<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/swap-nodes-in-pairs/' target='_blank'>LeetCode 24</a>

## Examples
**Example 1:** Input: head = [1,2,3,4], Output: [2,1,4,3]
