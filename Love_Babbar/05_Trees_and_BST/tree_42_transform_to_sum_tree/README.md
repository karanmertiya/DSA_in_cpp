# Tree 42 Transform To Sum Tree

## Description
Given a Binary Tree of size N , where each node can have positive or negative values. Convert this to a tree where each node contains the sum of the left and right sub trees of the original tree. The values of leaf nodes are changed to 0.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/transform-to-sum-tree/1' target='_blank'>GFG</a>
