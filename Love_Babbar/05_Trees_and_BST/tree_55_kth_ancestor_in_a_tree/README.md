# Tree 55 Kth Ancestor In A Tree

## Description
Given a binary tree of size N, a node, and a positive integer k. Your task is to find the kth ancestor of the given node in the binary tree. If there does not exist any such ancestor then return -1.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/kth-ancestor-in-a-tree/1' target='_blank'>GFG</a>
