# Dp 48 Maximum Length Chain Of Pairs

## Description
You are given N pairs of numbers. In every pair, the first number is always smaller than the second number. A pair (c, d) can follow another pair (a, b) if b < c. Chain of pairs can be formed in this fashion. You have to find the longest chain which can be formed from the given set of pairs.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/max-length-chain/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Sort and Greedy / DP.
