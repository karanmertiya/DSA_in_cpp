# DP 47 Egg Dropping Puzzle

## Description
You are given N identical eggs and you have access to a K-floored building from 1 to K. Find the minimum number of attempts you need in order to find the critical floor in the worst case.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/egg-dropping-puzzle-1587115620/1' target='_blank'>GFG</a>
