# Dp 29 Largest Divisible Subset

## Description
Given a set of distinct positive integers nums, return the largest subset answer such that every pair (answer[i], answer[j]) of elements in this subset satisfies: answer[i] % answer[j] == 0, or answer[j] % answer[i] == 0.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/largest-divisible-subset/' target='_blank'>LeetCode 368</a>

## Examples
<b>Example 1:</b> Sort and LIS variant.
