# DP 33 Longest Alternating Subsequence

## Description
A sequence {x1, x2, .. xn} is alternating sequence if its elements satisfy one of the following relations : x1 < x2 > x3 < x4 > x5..... or  x1 > x2 < x3 > x4 < x5..... Find the length of the longest alternating subsequence in an array of size N.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/longest-alternating-subsequence5951/1' target='_blank'>GFG</a>
