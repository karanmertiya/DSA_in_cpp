# Dp 38 Minimum Removals From Array To Make Max Min K

## Description
Given an array arr[] of size N and an integer K. The task is to find the minimum number of elements that should be removed, such that max(arr) - min(arr) <= K.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/minimum-removals3851/1' target='_blank'>GFG</a>
