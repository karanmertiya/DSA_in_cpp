# Graph 23 Word Ladder I

## Description
A transformation sequence from word beginWord to word endWord using a dictionary wordList is a sequence of words beginWord -> s1 -> s2 -> ... -> sk such that every adjacent pair of words differs by a single letter. Return the number of words in the shortest transformation sequence.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/word-ladder/' target='_blank'>LeetCode 127</a>
