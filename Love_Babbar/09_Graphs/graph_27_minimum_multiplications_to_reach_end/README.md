# Graph 27 Minimum Multiplications To Reach End

## Description
Given start, end and an array arr of n numbers. At each step, start is multiplied with any number in the array and then mod operation with 100000 is done to get the new start. Your task is to find the minimum steps in which end can be achieved starting from start. If it is not possible to reach end, then return -1.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/minimum-multiplications-to-reach-end/1' target='_blank'>GFG</a>

## Examples
**Example 1:** BFS / Dijkstra's with unit weights.
