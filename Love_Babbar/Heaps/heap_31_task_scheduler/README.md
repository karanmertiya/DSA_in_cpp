# heap_31_task_scheduler

Given a characters array tasks, representing the tasks a CPU needs to do, where each letter represents a different task. Tasks could be done in any order. Each task is done in one unit of time. For each unit of time, the CPU could complete either one task or just be idle. However, there is a non-negative integer n that represents the cooldown period between two same tasks. Return the least number of units of times that the CPU will take to finish all the given tasks.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/task-scheduler/' target='_blank'>LeetCode 621</a>
