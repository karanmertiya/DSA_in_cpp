# ll_47_clone_a_linked_list_with_next_and_random_pointer

You are given a special linked list with N nodes where each node has a next pointer pointing to its next node. You are also given M random pointers, where you will be given M number of pairs denoting two nodes a and b  i.e. a->arb = b (arb is pointer to random node). Construct a copy of the given list.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/clone-a-linked-list-with-next-and-random-pointer/1' target='_blank'>GFG</a>
