# Arr 58 Three Way Partitioning

## Description
Given an array of size n and a range [a, b]. The task is to partition the array around the range such that array is divided into three parts. 1) All elements smaller than a come first. 2) All elements in range a to b come next. 3) All elements greater than b appear in the end.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/three-way-partitioning/1' target='_blank'>GFG</a>
