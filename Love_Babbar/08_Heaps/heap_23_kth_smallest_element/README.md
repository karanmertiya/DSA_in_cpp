# Heap 23 Kth Smallest Element

## Description
Given an array arr[] and an integer K where K is smaller than size of array, the task is to find the Kth smallest element in the given array. It is given that all array elements are distinct.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/kth-smallest-element5635/1' target='_blank'>GFG</a>
