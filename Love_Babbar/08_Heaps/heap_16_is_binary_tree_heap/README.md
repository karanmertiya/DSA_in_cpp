# Heap 16 Is Binary Tree Heap

## Description
Given a binary tree, check if it has heap property or not. A binary tree is a max-heap if it is a complete binary tree and every node is greater than or equal to its children.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/is-binary-tree-heap/1' target='_blank'>GFG</a>
