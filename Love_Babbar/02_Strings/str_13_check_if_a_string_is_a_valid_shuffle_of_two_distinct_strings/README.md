# Str 13 Check If A String Is A Valid Shuffle Of Two Distinct Strings

## Description
Given strings str1 and str2. The task is to find if str3 is a valid shuffle of str1 and str2. A valid shuffle means that all the characters of str1 and str2 are present in str3 and their respective relative order is maintained.<br><br><b>🔗 Practice:</b> <a href='https://www.programiz.com/java-programming/examples/check-valid-shuffle-of-strings' target='_blank'>Article</a>
