# Str 18 Split The Binary String Into Substrings With Equal Number Of 0S And 1S

## Description
Given a binary string str of length N, the task is to find the maximum count of consecutive substrings str can be divided into such that all the substrings are balanced i.e. they have equal number of 0s and 1s. If it is not possible to split str satisfying the conditions then print -1.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/split-the-binary-string-into-substrings-with-equal-number-of-0s-and-1s/1' target='_blank'>GFG</a>
