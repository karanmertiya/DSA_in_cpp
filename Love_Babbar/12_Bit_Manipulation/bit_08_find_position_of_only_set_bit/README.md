# Bit 08 Find Position Of Only Set Bit

## Description
Given a number N having only one '1' and all other '0's in its binary representation, find position of the only set bit. If there are 0 or more than 1 set bit the answer should be -1. Position is 1-indexed from right to left.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/find-position-of-set-bit3706/1' target='_blank'>GFG</a>

## Examples
**Example 1:**
Input: N = 2
Output: 2
