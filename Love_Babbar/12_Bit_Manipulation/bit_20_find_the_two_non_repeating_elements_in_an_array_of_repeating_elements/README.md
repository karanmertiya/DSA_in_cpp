# Bit 20 Find The Two Non Repeating Elements In An Array Of Repeating Elements

## Description
Given an array in which all numbers except two are repeated once. (i.e. we have 2n+1 numbers and n numbers are occurring twice and remaining two have occurred once). Find those two numbers in the most efficient way.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/finding-the-numbers0215/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> XOR and rightmost set bit.
