# Bit 16 Find The Original Array Of Prefix Xor

## Description
You are given an integer array pref of size n. Find and return the array arr of size n that satisfies: pref[i] = arr[0] ^ arr[1] ^ ... ^ arr[i].<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/find-the-original-array-of-prefix-xor/' target='_blank'>LeetCode 2433</a>

## Examples
**Example 1:** `pref = [5,2,0,3,1]`. Output: `[5,7,2,3,2]`.
