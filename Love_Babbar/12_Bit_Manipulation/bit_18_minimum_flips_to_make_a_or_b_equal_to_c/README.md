# Bit 18 Minimum Flips To Make A Or B Equal To C

## Description
Given 3 positives numbers a, b and c. Return the minimum flips required in some bits of a and b to make ( a OR b == c ). (bitwise OR operation).<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/minimum-flips-to-make-a-or-b-equal-to-c/' target='_blank'>LeetCode 1318</a>

## Examples
**Example 1:**

**Input:** a=2, b=6, c=5
**Output:** 3
