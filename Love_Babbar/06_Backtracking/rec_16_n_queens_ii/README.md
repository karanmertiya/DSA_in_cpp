# Rec 16 N Queens Ii

## Description
The n-queens puzzle is the problem of placing n queens on an n x n chessboard such that no two queens attack each other. Given an integer n, return the number of distinct solutions to the n-queens puzzle.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/n-queens-ii/' target='_blank'>LeetCode 52</a>
