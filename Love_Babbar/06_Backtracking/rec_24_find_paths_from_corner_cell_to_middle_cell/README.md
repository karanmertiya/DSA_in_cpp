# Rec 24 Find Paths From Corner Cell To Middle Cell

## Description
Given a square matrix of size N x N. The task is to find all paths from corner cells to the middle cell. The number of steps you can take is the value of the cell.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/paths-from-corner-to-middle/0' target='_blank'>GFG</a>

## Examples
```text
<b>Example 1:</b> BFS / DFS for path finding.
```
