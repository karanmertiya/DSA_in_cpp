# bm_20_decode_xored_array

There is a hidden integer array arr that consists of n non-negative integers. It was encoded into another integer array encoded of length n - 1, such that encoded[i] = arr[i] XOR arr[i + 1]. You are also given the first element arr[0]. Return the original array arr.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/decode-xored-array/' target='_blank'>LeetCode 1720</a>
