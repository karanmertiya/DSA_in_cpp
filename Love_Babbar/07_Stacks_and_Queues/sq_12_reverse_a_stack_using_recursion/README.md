# Sq 12 Reverse A Stack Using Recursion

## Description
Write a program to reverse a stack using recursion. You are not allowed to use loop constructs like while, for..etc, and you can only use the following ADT functions on Stack S: isEmpty(S), push(S), pop(S).<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/reverse-a-stack/1' target='_blank'>GFG</a>

## Examples
**Example 1:** Recursive pop and insertAtBottom.
