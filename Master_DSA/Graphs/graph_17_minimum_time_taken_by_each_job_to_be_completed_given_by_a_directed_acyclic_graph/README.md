# graph_17_minimum_time_taken_by_each_job_to_be_completed_given_by_a_directed_acyclic_graph

Given a Directed Acyclic Graph having V vertices and E edges, where each edge {U, V} represents the Jobs U and V such that Job V can only be started only after completion of Job U.
