# graph_56_find_closest_node_to_given_two_nodes

You are given a directed graph of n nodes numbered from 0 to n - 1, where each node has at most one outgoing edge. You are given an integer array edges of length n. Return the index of the node that can be reached from both node1 and node2, such that the maximum between the distance from node1 to that node, and from node2 to that node is minimized.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/find-closest-node-to-given-two-nodes/' target='_blank'>LeetCode 2359</a>
