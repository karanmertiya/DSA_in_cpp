# graph_21_dijkstras_algorithm

Given a weighted, undirected and connected graph of V vertices and an adjacency list adj where adj[i] is a list of lists containing two integers where the first integer of each list j denotes there is edge between i and j, second integers corresponds to the weight of that edge.
