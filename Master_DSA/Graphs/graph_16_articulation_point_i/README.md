# Graph 16 Articulation Point I

## Description
Given an undirected connected graph with V vertices and E edges, return all the articulation points of the graph. An articulation point (or cut vertex) is any node whose removal increases the number of connected components.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/articulation-point-1/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Tarjan's algorithm with discovery times.
