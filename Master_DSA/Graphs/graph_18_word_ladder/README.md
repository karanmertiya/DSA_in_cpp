# graph_18_word_ladder

A transformation sequence from word beginWord to word endWord using a dictionary wordList is a sequence of words beginWord -> s1 -> s2 -> ... -> sk such that every adjacent pair differs by a single letter, and every si is in wordList. Given two words, beginWord and endWord, and a dictionary wordList, return the number of words in the shortest transformation sequence from beginWord to endWord, or 0 if no such sequence exists.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/word-ladder/' target='_blank'>LeetCode 127</a>
