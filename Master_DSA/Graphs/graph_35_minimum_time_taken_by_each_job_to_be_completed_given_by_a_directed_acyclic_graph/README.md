# Graph 35 Minimum Time Taken By Each Job To Be Completed Given By A Directed Acyclic Graph

## Description
Given a Directed Acyclic Graph having V vertices and E edges, where each edge {U, V} represents the Jobs U and V such that Job V can only be started only after completion of Job U. Find the minimum time taken by each job to be completed where each Job takes unit time to get completed.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/minimum-time-taken-by-each-job-to-be-completed-given-by-a-directed-acyclic-graph/1' target='_blank'>GFG</a>
