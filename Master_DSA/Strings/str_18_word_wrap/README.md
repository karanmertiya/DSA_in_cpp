# Str 18 Word Wrap

## Description
Given an array nums[] of size n, where nums[i] denotes the number of characters in one word. Let K be the limit on the number of characters that can be put in one line. Put line breaks so that the lines are printed neatly. Return the minimum total cost.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/word-wrap1646/1' target='_blank'>GFG</a>
