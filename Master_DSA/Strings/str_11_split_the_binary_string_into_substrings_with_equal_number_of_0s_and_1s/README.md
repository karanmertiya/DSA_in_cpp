# str_11_split_the_binary_string_into_substrings_with_equal_number_of_0s_and_1s

Given a binary string str of length N, the task is to find the maximum count of consecutive substrings str can be divided into such that all the substrings are balanced i.e. they have equal number of 0s and 1s.
