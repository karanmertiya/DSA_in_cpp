# arr_24_reverse_pairs

Given an integer array nums, return the number of reverse pairs in the array. A reverse pair is a pair (i, j) where 0 <= i < j < nums.length and nums[i] > 2 * nums[j].<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/reverse-pairs/' target='_blank'>LeetCode 493</a>
