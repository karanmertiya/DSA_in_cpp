# Arr 46 Maximum Trains For Which Stoppage Can Be Provided

## Description
We are given n-trains with their arrival time, departure time and platform number. The task is to find the maximum number of trains that can be stopped at a given number of platforms such that no two trains overlap on the same platform.<br><br><b>🔗 Practice:</b> <a href='https://www.geeksforgeeks.org/maximum-trains-stoppage-can-provided/' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Activity Selection on each platform.
