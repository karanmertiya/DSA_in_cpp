# Arr 49 Greedy 10 Shop In Candy Store

## Description
In a candy store, there are N different types of candies available and the prices of all the N different types of candies are provided. You are given an offer: if you buy one candy you can get K other candies for free. Find the minimum and maximum amount to buy all candies.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/shop-in-candy-store1145/1' target='_blank'>GFG</a>
