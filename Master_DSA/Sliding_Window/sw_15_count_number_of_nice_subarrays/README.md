# Sw 15 Count Number Of Nice Subarrays

## Description
Given an array of integers nums and an integer k. A continuous subarray is called nice if there are k odd numbers on it. Return the number of nice sub-arrays.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/count-number-of-nice-subarrays/' target='_blank'>LeetCode 1248</a>

## Examples
**Example 1:** atMost(k) - atMost(k - 1).
