# Two Pointers & Sliding Window
## Core Concept
An optimization technique used on Arrays and Strings. A window is formed over a subset of data and moves accordingly to find sub-ranges.
## Syntax & Initialization
Usually requires two pointers `left = 0` and `right = 0` and a frequency map or sum tracker.
## Main Operations
- Expanding Window: `O(1)` per step
- Shrinking Window: `O(1)` per step (Amortized `O(N)` total)
