# dp_59_maximum_difference_of_zeros_and_ones_in_binary_string

Given a binary string S consisting of 0s and 1s. The task is to find the maximum difference of the number of 0s and the number of 1s (number of 0s - number of 1s) in the substrings of a string. Note: In the case of all 1s, the answer will be -1.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/maximum-difference-of-zeros-and-ones-in-binary-string4111/1' target='_blank'>GFG</a>
