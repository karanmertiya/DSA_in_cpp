# Dp 42 Palindromic Partitioning

## Description
Given a string str, a partitioning of the string is a palindrome partitioning if every sub-string of the partition is a palindrome. Determine the fewest cuts needed for palindrome partitioning of given string.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/palindromic-patitioning4845/1' target='_blank'>GFG</a>
