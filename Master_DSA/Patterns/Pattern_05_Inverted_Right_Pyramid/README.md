# Pattern 05 Inverted Right Pyramid

**Tags:** `Striver A Z` | `Love Babbar` | `Apna College`

## Problem Description
Given an integer N, print an inverted right-angled triangle pattern of '*'.

## Examples
```text
Example 1:
Input: N = 3
Output:
* * *
* *
*

Example 2:
Input: N = 5
Output:
* * * * *
* * * *
* * *
* *
*
```
