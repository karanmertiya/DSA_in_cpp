# Pattern 01 Rectangular Star

**Tags:** `Striver A Z` | `Love Babbar` | `Apna College`

## Problem Description
Given an integer N, print a rectangular pattern of N rows and N columns consisting of '*'.

## Examples
```text
Example 1:
Input: N = 3
Output:
***
***
***

Example 2:
Input: N = 5
Output:
*****
*****
*****
*****
*****
```
