# Pattern 06 Inverted Numbered Right Pyramid

**Tags:** `Striver A Z` | `Love Babbar` | `Apna College`

## Problem Description
Given an integer N, print an inverted right-angled triangle pattern of numbers.

## Examples
```text
Example 1:
Input: N = 3
Output:
1 2 3
1 2
1

Example 2:
Input: N = 5
Output:
1 2 3 4 5
1 2 3 4
1 2 3
1 2
1
```
