# Pattern 02 Right Angled Triangle

**Tags:** `Striver A Z` | `Love Babbar` | `Apna College`

## Problem Description
Given an integer N, print a right-angled triangle pattern of '*' with N rows.

## Examples
```text
Example 1:
Input: N = 3
Output:
*
**
***

Example 2:
Input: N = 5
Output:
*
**
***
****
*****
```
