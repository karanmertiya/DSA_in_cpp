# Pattern 08 Inverted Star Pyramid

**Tags:** `Striver A Z` | `Love Babbar` | `Apna College`

## Problem Description
Given an integer N, print an inverted full pyramid pattern of '*'.

## Examples
```text
Example 1:
Input: N = 3
Output:
*****
 ***
  *

Example 2:
Input: N = 5
Output:
*********
 *******
  *****
   ***
    *
```
