# Rec 32 Find If There Is A Path Of More Than K Length From A Source

## Description
Given a graph, a source vertex and a number k, find if there is a simple path (without any cycle) starting from given source and ending at any other vertex such that the distance from source to that vertex is at least 'k' length.<br><br><b>🔗 Practice:</b> <a href='https://www.geeksforgeeks.org/find-if-there-is-a-path-of-more-than-k-length-from-a-source/' target='_blank'>GFG</a>
