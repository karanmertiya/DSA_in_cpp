# Rec 33 Longest Possible Route In A Matrix With Hurdles

## Description
Given an M x N matrix, with a few hurdles(denoted by 0) arbitrarily placed, calculate the length of the longest possible route possible from source to a destination within the matrix. We are allowed to move to only adjacent cells which are not hurdles. The route cannot contain any diagonal moves and a location once visited in a particular path cannot be visited again.<br><br><b>🔗 Practice:</b> <a href='https://www.geeksforgeeks.org/longest-possible-route-in-a-matrix-with-hurdles/' target='_blank'>GFG</a>
