# Rec 39 Find All Factorial Numbers Less Than Or Equal To N

## Description
A number N is called a factorial number if it is the factorial of a positive integer. For example, the first few factorial numbers are 1, 2, 6, 24, 120. Given a number N, the task is to return the list/vector of the factorial numbers smaller than or equal to N.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/find-all-factorial-numbers-less-than-or-equal-to-n3548/1' target='_blank'>GFG</a>
