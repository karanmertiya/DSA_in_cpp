# Rec 30 Combinational Sum

## Description
Given an array of integers arr and an integer sum, find all unique combinations in arr where the sum is equal to sum. The same repeated number may be chosen from arr unlimited number of times. Elements in a combination must be printed in non-decreasing order.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/combination-sum-1587115620/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Backtracking.
