# Rec 42 Count Good Numbers

## Description
A digit string is good if the digits (0-indexed) at even indices are even and the digits at odd indices are prime (2, 3, 5, or 7). Given an integer n, return the total number of good digit strings of length n. Since the answer may be large, return it modulo 10^9 + 7.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/count-good-numbers/' target='_blank'>LeetCode 1922</a>
