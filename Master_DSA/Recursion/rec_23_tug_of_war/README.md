# Rec 23 Tug Of War

## Description
Given a set of n integers, divide the set in two subsets of n/2 sizes each such that the absolute difference of the sum of two subsets is as minimum as possible.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/tug-of-war/0' target='_blank'>GFG</a>
