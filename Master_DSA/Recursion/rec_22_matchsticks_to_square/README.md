# Rec 22 Matchsticks To Square

## Description
You are given an integer array matchsticks where matchsticks[i] is the length of the ith matchstick. You want to use all the matchsticks to make one square. Return true if you can make this square and false otherwise.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/matchsticks-to-square/' target='_blank'>LeetCode 473</a>
