# tree_27_binary_tree_to_cdll

Given a Binary Tree of N edges. The task is to convert this to a Circular Doubly Linked List (CDLL) in-place. The left and right pointers in nodes are to be used as previous and next pointers respectively in converted CDLL. The order of nodes in CDLL must be same as Inorder of the given Binary Tree.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/binary-tree-to-cdll/1' target='_blank'>GFG</a>
