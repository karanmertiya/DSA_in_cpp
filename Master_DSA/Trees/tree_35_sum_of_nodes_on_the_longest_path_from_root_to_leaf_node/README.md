# tree_35_sum_of_nodes_on_the_longest_path_from_root_to_leaf_node

Given a binary tree of size N. Your task is to complete the function sumOfLongRootToLeafPath(), that find the sum of all nodes on the longest path from root to leaf node. If two or more paths compete for the longest path, then the path having maximum sum of nodes is being considered.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/sum-of-the-longest-bloodline-of-a-tree/1' target='_blank'>GFG</a>
