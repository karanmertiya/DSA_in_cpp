# Tree 53 Check If 2 Trees Are Mirror Or Not

## Description
Given two n-ary trees. Check if they are mirror images of each other or not. You are also given e denoting the number of edges in both trees, and two arrays, A[] and B[]. Each array has 2*e space separated values u,v denoting an edge from u to v for the both trees.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/check-mirror-in-n-ary-tree1528/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Stack and Queue.
