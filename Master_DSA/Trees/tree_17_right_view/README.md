# Tree 17 Right View

## Description
Given the root of a binary tree, imagine yourself standing on the right side of it, return the values of the nodes you can see ordered from top to bottom.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/binary-tree-right-side-view/' target='_blank'>LeetCode 199</a>

## Examples
**Example 1:** Input: root = [1,2,3,null,5,null,4], Output: [1,3,4]
