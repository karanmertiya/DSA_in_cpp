# Bit 12 Number Of Steps To Reduce A Number To Zero

## Description
Given an integer num, return the number of steps to reduce it to zero. In one step, if the current number is even, you have to divide it by 2, otherwise, you have to subtract 1 from it.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/number-of-steps-to-reduce-a-number-to-zero/' target='_blank'>LeetCode 1342</a>
