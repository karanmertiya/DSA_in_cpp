# Bit 14 Count Total Set Bits 1 To N

## Description
You are given a number N. Find the total count of set bits for all numbers from 1 to N(both inclusive).<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/count-total-set-bits-1587115620/1' target='_blank'>GFG</a>

## Examples
**Example 1:**

**Input:** N=4
**Output:** 5
Explanation: 1(01) + 2(10) + 3(11) + 4(100) -> 1+1+2+1 = 5
