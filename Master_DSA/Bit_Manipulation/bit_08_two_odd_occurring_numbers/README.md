# Bit 08 Two Odd Occurring Numbers

## Description
Given an array in which all numbers appear exactly twice except two numbers that appear only once. Find the two numbers.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/two-numbers-with-odd-occurrences5846/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Input: Arr = {4, 2, 4, 5, 2, 3, 3, 1}, Output: {5, 1}
