# Bit Manipulation

## Core Concept
Operating directly on hardware-level binary integers to achieve highly optimized spatial and time complexities.

## Syntax & Initialization
**C++:** Logical bitwise `&`, `|`, `^`, `~`, `<<`, `>>`. Unsigned `uint32_t` heavily used.
**Python:** Identical operators `&`, `|`, `^`, `~`, `<<`, `>>`. 

## Main Operations
- Set/Clear/Toggle Bit: `O(1)` Time
- Count Set Bits: `O(Set Bits)` Time via Brian Kernighan
