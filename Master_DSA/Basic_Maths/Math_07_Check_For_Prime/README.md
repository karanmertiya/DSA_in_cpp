# Math 07 Check For Prime

**Tags:** `Striver A Z` | `Love Babbar` | `Apna College`

## Problem Description
Given an integer N, check whether it is prime or not. A prime number is only divisible by 1 and itself.

## Examples
```text
Example 1:
Input: N = 11
Output: true

Example 2:
Input: N = 15
Output: false
```
