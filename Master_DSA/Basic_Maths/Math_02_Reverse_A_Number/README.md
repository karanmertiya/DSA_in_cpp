# Math 02 Reverse A Number

**Tags:** `Striver A Z` | `Love Babbar` | `Apna College`

## Problem Description
Given an integer N, return the reverse of the given number. Note: if a number has trailing zeros, then its reverse will not include them.

## Examples
```text
Example 1:
Input: N = 1234
Output: 4321

Example 2:
Input: N = 10400
Output: 401
```
