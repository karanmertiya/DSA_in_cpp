# Math 03 Check Palindrome

**Tags:** `Striver A Z` | `Love Babbar` | `Apna College`

## Problem Description
Given an integer N, check if it is a palindrome. A palindrome reads the same forwards and backwards.

## Examples
```text
Example 1:
Input: N = 121
Output: true

Example 2:
Input: N = 123
Output: false
```
