# Math 05 Armstrong Numbers

**Tags:** `Striver A Z` | `Apna College`

## Problem Description
Given an integer N, check if it is an Armstrong number. An Armstrong number is equal to the sum of its own digits each raised to the power of the total number of digits.

## Examples
```text
Example 1:
Input: N = 153
Output: true

Example 2:
Input: N = 170
Output: false
```
