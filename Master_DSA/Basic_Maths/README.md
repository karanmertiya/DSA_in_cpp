# Basic Maths
## Core Concept
Mathematical algorithms for extracting digits, determining prime boundaries, and finding greatest common divisors while preventing overflow constraints.
## Syntax & Initialization
**C++:** Native 32-bit `int` or 64-bit `long long`. Include `<cmath>` for `sqrt()`, `abs()`.
**Python:** Native arbitrary-precision `int`. Import `math` for `math.sqrt()`, `math.gcd()`.
## Main Operations
- Modulo Extraction: `O(1)` Time
- Traversal via Division: `O(log10(N))` Time
