# tries_19_longest_word_in_dictionary

Given an array of strings words representing an English Dictionary, return the longest word in words that can be built one character at a time by other words in words. If there is more than one possible answer, return the longest word with the smallest lexicographical order.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/longest-word-in-dictionary/' target='_blank'>LeetCode 720</a>
