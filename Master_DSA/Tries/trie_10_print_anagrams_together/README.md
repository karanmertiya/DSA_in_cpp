# Trie 10 Print Anagrams Together

## Description
Given an array of strings, return all groups of strings that are anagrams. The groups must be created in order of their appearance in the original array. Solve using a Trie.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/print-anagrams-together/1' target='_blank'>GFG</a>
