# Trie 07 Maximum Xor With An Element From Array

## Description
You are given an array nums consisting of non-negative integers. You are also given a queries array, where queries[i] = [xi, mi]. The answer to the ith query is the maximum bitwise XOR value of xi and any element of nums that does not exceed mi. If all elements in nums are larger than mi, then the answer is -1.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/maximum-xor-with-an-element-from-array/' target='_blank'>LeetCode 1707</a>
