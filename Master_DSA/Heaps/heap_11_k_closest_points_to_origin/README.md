# Heap 11 K Closest Points To Origin

## Description
Given an array of points where points[i] = [xi, yi] represents a point on the X-Y plane and an integer k, return the k closest points to the origin (0, 0).<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/k-closest-points-to-origin/' target='_blank'>LeetCode 973</a>

## Examples
<b>Example 1:</b> Max-heap of pairs.
