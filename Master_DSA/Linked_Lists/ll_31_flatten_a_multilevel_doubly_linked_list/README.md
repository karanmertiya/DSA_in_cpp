# Ll 31 Flatten A Multilevel Doubly Linked List

## Description
You are given a doubly linked list, which contains nodes that have a next pointer, a previous pointer, and an additional child pointer. Flatten the list so that all the nodes appear in a single-level, doubly linked list.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/flatten-a-multilevel-doubly-linked-list/' target='_blank'>LeetCode 430</a>
