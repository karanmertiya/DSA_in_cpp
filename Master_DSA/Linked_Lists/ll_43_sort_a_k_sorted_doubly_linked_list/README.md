# Ll 43 Sort A K Sorted Doubly Linked List

## Description
Given a doubly linked list containing N nodes, where each node is at most k away from its target position in the sorted list. The problem is to sort the given doubly linked list.<br><br><b>🔗 Practice:</b> <a href='https://www.geeksforgeeks.org/sort-k-sorted-doubly-linked-list/' target='_blank'>GFG</a>
