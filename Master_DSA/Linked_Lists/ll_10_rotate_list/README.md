# Ll 10 Rotate List

## Description
Given the head of a linked list, rotate the list to the right by k places.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/rotate-list/' target='_blank'>LeetCode 61</a>

## Examples
```text
<b>Example 1:</b> Input: head = [1,2,3,4,5], k = 2, Output: [4,5,1,2,3]
```
