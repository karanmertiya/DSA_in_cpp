# Ll 22 Partition List

## Description
Given the head of a linked list and a value x, partition it such that all nodes less than x come before nodes greater than or equal to x. You should preserve the original relative order.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/partition-list/' target='_blank'>LeetCode 86</a>

## Examples
<b>Example 1:</b> Two lists then join.
