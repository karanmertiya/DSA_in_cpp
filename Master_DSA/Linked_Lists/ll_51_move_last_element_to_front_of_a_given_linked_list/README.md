# Ll 51 Move Last Element To Front Of A Given Linked List

## Description
Write a function that moves the last element to the front in a given Singly Linked List. For example, if the given Linked List is 1->2->3->4->5, then the function should change the list to 5->1->2->3->4.<br><br><b>🔗 Practice:</b> <a href='https://www.geeksforgeeks.org/move-last-element-to-front-of-a-given-linked-list/' target='_blank'>GFG</a>
