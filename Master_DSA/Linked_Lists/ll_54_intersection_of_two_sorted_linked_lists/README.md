# Ll 54 Intersection Of Two Sorted Linked Lists

## Description
Given two lists sorted in increasing order, create a new list representing the intersection of the two lists. The new list should be made with its own memory — the original lists should not be changed.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/intersection-of-two-sorted-linked-lists/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Two Pointers.
