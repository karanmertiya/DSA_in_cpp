# graph_03_rotting_oranges

You are given an m x n grid where each cell can have 0 (empty), 1 (fresh), or 2 (rotten). Every minute, any fresh orange that is 4-directionally adjacent to a rotten orange becomes rotten. Return the minimum number of minutes.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/rotting-oranges/' target='_blank'>LeetCode 994</a>
