# arr_61_3sum

Given an integer array nums, return all the triplets [nums[i], nums[j], nums[k]] such that i != j, i != k, and j != k, and nums[i] + nums[j] + nums[k] == 0. Notice that the solution set must not contain duplicate triplets.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/3sum/' target='_blank'>LeetCode 15</a>
