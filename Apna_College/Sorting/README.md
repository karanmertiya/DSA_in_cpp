# Sorting Algorithms
## Core Concept
Algorithms to organize linear data sequentially. Essential prerequisite for Binary Search.
## Syntax & Initialization
`std::sort()` in C++ and `.sort()` in Python utilize hybrid algorithms (Introsort/Timsort) which run in `O(N log N)`.
## Main Operations
- Best Case `O(N)` (Bubble sort with flags)
- Worst Case `O(N^2)` (Selection, Bubble, Insertion)
- Optimal `O(N log N)` (Merge, Quick, Heap)
