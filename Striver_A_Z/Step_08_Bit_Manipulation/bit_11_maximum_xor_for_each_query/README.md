# Bit 11 Maximum Xor For Each Query

## Description
You are given a sorted array nums of n non-negative integers and an integer maximumBit. You want to perform the following query n times: Find a non-negative integer k < 2^maximumBit such that nums[0] XOR ... XOR nums[nums.length-1] XOR k is maximized. Then remove the last element from the current array.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/maximum-xor-for-each-query/' target='_blank'>LeetCode 1829</a>

## Examples
**Example 1:** `nums = [0,1,1,3], maximumBit = 2`. Output: `[0,3,2,3]`.
