# str_40_count_and_say

The count-and-say sequence is a sequence of digit strings defined by the recursive formula: countAndSay(1) = "1". countAndSay(n) is the way you would "say" the digit string from countAndSay(n-1), which is then converted into a different digit string.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/count-and-say/' target='_blank'>LeetCode 38</a>
