# Rec 10 Palindrome Partitioning

## Description
Given a string s, partition s such that every substring of the partition is a palindrome. Return all possible palindrome partitioning of s.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/palindrome-partitioning/' target='_blank'>LeetCode 131</a>

## Examples
```text
<b>Example 1:</b> Input: s = 'aab', Output: [['a','a','b'],['aa','b']]
```
