# Tree 41 Convert Binary Tree Into Doubly Linked List

## Description
Given a Binary Tree (BT), convert it to a Doubly Linked List(DLL) In-Place. The left and right pointers in nodes are to be used as previous and next pointers respectively in converted DLL. The order of nodes in DLL must be same as Inorder of the given Binary Tree.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/binary-tree-to-dll/1' target='_blank'>GFG</a>

## Examples
**Example 1:** DFS Inorder, maintaining a `prev` pointer.
