# Str 19 Maximum Nesting Depth Of The Parentheses

## Description
A string is a valid parentheses string (denoted VPS) if it meets one of the following: It is an empty string "", or a single character not equal to "(" or ")", It can be written as AB, or It can be written as (A), where A is a VPS. Given a VPS represented as string s, return the nesting depth of s.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/maximum-nesting-depth-of-the-parentheses/' target='_blank'>LeetCode 1614</a>
