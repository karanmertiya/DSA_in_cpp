# Graph 17 Number Of Provinces Dsu

## Description
There are n cities. Some of them are connected, while some are not. If city a is connected directly with city b, and city b is connected directly with city c, then city a is connected indirectly with city c. A province is a group of directly or indirectly connected cities and no other cities outside of the group. Find number of provinces using Disjoint Set Union (DSU).<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/number-of-provinces/' target='_blank'>LeetCode 547</a>

## Examples
<b>Example 1:</b> Connect elements, count unique parents.
