# Graph 12 Alien Dictionary

## Description
Given a sorted dictionary of an alien language having N words and k starting alphabets of standard dictionary. Find the order of characters in the alien language.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/alien-dictionary/1' target='_blank'>GFG</a>

## Examples
**Example 1:** words = ['baa', 'abcd', 'abca', 'cab', 'cad'], K = 4, Output: 'bdac'
