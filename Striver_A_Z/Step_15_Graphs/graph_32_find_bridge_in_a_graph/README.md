# Graph 32 Find Bridge In A Graph

## Description
Given a Graph of V vertices and E edges and another edge(c - d), the task is to find if the given edge is a Bridge. i.e., removing the edge disconnects the graph.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/bridge-edge-in-graph/1' target='_blank'>GFG</a>
