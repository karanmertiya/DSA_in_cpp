# Sw 06 Longest K Unique Characters Substring

## Description
Given a string you need to print the size of the longest possible substring that has exactly K unique characters. If there is no possible substring then print -1.<br><br><b>🔗 Practice:</b> <a href='https://practice.geeksforgeeks.org/problems/longest-k-unique-characters-substring0853/1' target='_blank'>GFG</a>

## Examples
<b>Example 1:</b> Variable window and hash map.
