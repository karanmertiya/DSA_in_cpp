# Sw 15 Find All Anagrams In A String

## Description
Given two strings s and p, return an array of all the start indices of p's anagrams in s. You may return the answer in any order.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/find-all-anagrams-in-a-string/' target='_blank'>LeetCode 438</a>
