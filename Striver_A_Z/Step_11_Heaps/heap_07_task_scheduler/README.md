# Heap 07 Task Scheduler

## Description
Given a characters array tasks, representing the tasks a CPU needs to do, and a non-negative integer n that represents the cooldown period between two same tasks, return the least number of units of times that the CPU will take to finish all the given tasks.<br><br><b>🔗 Practice:</b> <a href='https://leetcode.com/problems/task-scheduler/' target='_blank'>LeetCode 621</a>

## Examples
<b>Example 1:</b> Greedy placement.
